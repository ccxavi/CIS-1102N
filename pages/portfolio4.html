Computer Hardware and Software

Introduction


  • What is a Computer?:
    • A computer consists of hardware and software.
    • Hardware: Physical components (input, processing, output, and storage devices).
    • Software: Instructions (e.g., Windows XP and other programs) that enable hardware to function and allow user interaction.
    • Hardware is tangible and visible; software enables its operation.
  • Computer Systems:
    • A computer system integrates hardware and software to process data and generate meaningful results.
    • Performs basic functions:
      • Input
      • Processing
      • Storage
      • Output
      • Data transmission
    • Data and instructions are entered, processed, stored for later use, and output in a useful format.
    • Connected to a network for data transmission and information sharing.


Types of Computer Systems


  • Supercomputers
    • Most advanced, powerful, and expensive computers of the day
    • Characterized as having the fastest processing speeds and performing the most complex calculations
    • Widely used in scientific research, artificial intelligence, defense systems, and industrial design
  • Mainframe Computer
    • An advanced multi-user machine typically used to manage databases, financial transactions, and communications of large organizations such as banks, hospitals, retail stores, insurance companies, and government offices
    • Can process billions of instructions per second, support hundreds of users, and store terabytes (trillions of bytes) of data
  • Personal Computer
    • A system that uses a microprocessor to provide computing to a single user
    • Developed in 1975 by computing enthusiasts who wanted their own computer rather than share a large centralized mainframe
    • Has many different names and configurations including microcomputer, laptop, desktop, and tablet


Computer Hardware


Hardware of a computer is made up of complex electronic circuits. For a user, the details of the circuitry are not important. However, the hardware units with which a user has to interact must be clearly understood. For convenience, the hardware of a computer can be classified in the following categories:


  • Input Devices:
    • Data is not always in a computer-friendly format, so input devices are needed to convert it.
    • These devices take user data and instructions, transform them into a form the computer can process, and send it to the system.
    • For example, to enter employees' names, you'd use an input device—not paper.
  • Output Devices:
    • A computer's processed output isn't always user-readable, so output devices convert it into a form users can understand.
  • Central Processing Unit (CPU):
    • The CPU is the computer's "brain," essential for its function.
    • It consists of the Arithmetic Logic Unit (ALU), which handles calculations, and the Control Unit (CU), which manages other hardware.
    • CPUs excel in both speed and control.
  • Memory or Storage:
    • This hardware stores all data and instructions for a computer, including processing results.
    • Computers have different memory types, with some directly linked to the CPU, offering very fast data access. These are called primary memory—such as RAM (Random Access Memory) and ROM (Read Only Memory).
    • The CPU uses data and instructions only from primary memory.
    • Primary memories come in types: volatile memory, like RAM, loses its contents when power is off, while non-volatile memory, like ROM, retains data without power.
    • Secondary storage devices, such as hard disks, floppy disks, and CD-ROMs, are external and portable.
    • Devices like input, output, and secondary storage are not directly linked to the CPU, so they are called peripheral devices.
  • Motherboard:
    • The motherboard, also known as the system board, is the main hardware component of a microcomputer.
    • It connects all other circuit boards with chips and electronic parts, acting as the central "mother" to all components.


Cental Processing Unit


The Central Processing Unit (CPU) is the computer's core processor, executing instructions and processing data. In microcomputers, the CPU, known as a microprocessor, is a single-chip component. Together with memory, it forms the central system that connects to all peripherals.


The CPU consists of the following distinct parts:


  • Arithmetic Logic Unit (ALU)
    • The CPU's arithmetic and logic unit (ALU) handles all arithmetic operations (addition, subtraction, multiplication, division) and logical comparisons (less than, equal to, greater than).
  • Control Unit
    • The control unit manages data and instruction flow between computer units, coordinating all operations like a "Central Nervous System."
    • It retrieves, interprets, and directs instructions and handles data transfer between memory and peripherals.
  • Registers
    • Registers are fast memory circuits that temporarily store data, instructions, or addresses for the ALU during operations.
    • Each holds one word (2 bytes, or 16 bits) until replaced.
    • Their type and number vary by CPU, with categories including General Purpose, Pointer, Segment, Index, Flags, and Instruction Pointer Registers.
  • Buses
    • Data is stored in registers as eight-bit units, where each bit (0 or 1) is transferred via individual wires.
    • A group of eight wires used to transfer data between registers is called a bus.
    • A bus connects components to transmit signals and comes in three types:
      • Data Bus (moves data)
      • Address Bus (transfers memory addresses)
      • Control Bus (sends control signals)
  • Clock
    • The CPU clock allocates a set time for each micro-operation, synchronizing the CPU's actions.
    • Clock speed, measured in MHz (millions of cycles per second), varies by model, ranging from 4.77 MHz in the 8088 processor to 266 MHz in the Pentium II.
    • CPU speed can also be measured in MIPS (Millions of Instructions Per Second) or MFLOPS (Millions of Floating Point Operations Per Second).


Input Devices


Input devices are used to enter data, information, and instructions into RAM. They typically function as standalone data entry units with a small processor, keyboard, and display. The processor ensures data accuracy during input.


The common input devices are the following:


  • Keyboard: The keyboard is a common computer input device with a QWERTY layout similar to a typewriter. It typically has three sections: typing keys, cursor movement keys, and a numeric keypad. The Num Lock key activates the numeric keypad for quick number entry. Standard keyboards are flat and include special function keys, arrow keys, and cursor controls.
  • Mouse: When using a display, we often point to select options and navigate on the screen. A mouse allows smooth cursor movement; an optical mouse is quiet and reliable, while a mechanical one is more affordable. In Windows, users perform “point-and-click” actions by aiming the pointer, moving the mouse, and clicking or double-clicking as needed. Laptops may use touchpads, trackballs, or other pointer devices with extra click keys. Wrist pads can help reduce strain during extended use.
  • Trackball: A trackball resembles a mouse but has a roller on top and buttons on the side. It moves the cursor by spinning the ball in the desired direction. Often preferred for requiring minimal arm movement and space, it is commonly used with portable computers.
  • Light Pen: A light pen is a pen-shaped pointing device for selecting items or drawing on a screen. It has a light receptor that activates when pressed against the display, detecting its position. A photocell and optical system in the pen send signals to the CPU, enabling responsive actions on the screen.
  • Touch Screen: A touch screen responds to finger taps, allowing users to select items directly on the screen without pressing keys.
  • Game Controller: A game controller is used with gaming systems to control characters or provide input. It connects to consoles or computers via wired or wireless methods and includes devices like keyboards, mice, gamepads, and joysticks.
  • Microphone: A microphone, or mic, is a hardware device that lets users input audio into their computers.


Ouput Devices


Once data is processed, output devices translate the language of bits into a form humans can understand. Output devices are divided into two basic categories: those that produce hard copy, including printers and plotters; and those that produce soft (digital) copy, including monitors (the most commonly used output devices. The output normally can be produced in two ways - either on a display unit/device or on a paper. Other kinds of output, such as speech output, are also being used in certain applications.


  • Monitors: A Visual Display Unit (VDU), or monitor, is a computer's primary output device. It uses a Cathode Ray Tube (CRT) to show characters and forms images from small dots, called pixels. Image clarity, or resolution, depends on pixel quantity.
  • Video Controller: A video adapter, also called a graphics card or video controller, is an internal circuit board that enables a display device like a monitor to show images from the computer.
  • PC Projector: A projector uses light and lenses to enlarge and display images on a larger screen or wall, whether still or moving, depending on its design.
  • Speakers: A hardware device linked to a computer's sound card that produces the sounds it generates.
  • Printer: Printers produce output on paper and come in various types, classified by print quality and speed.


Storage Devices


  • Magnetic Storage Devices: This storage device has a magnetic coating on a rigid or flexible surface, with a read-write head that converts data and instructions into magnetic signals, which are then stored on the medium.
    • Floppy Disks:
      • IBM introduced the floppy disk, initially 8 inches in diameter.
      • Over time, it became smaller, eventually becoming the 3.5-inch diskette, which holds 1.44 MB of data.
      • This size is widely used in microcomputers for storing programs and data, equivalent to about 400 pages of a printed book.
      • Floppy disks also feature a write-protect notch with a sliding switch.
    • Hard Disks:
      • Hard disks, introduced in the 1950s, initially had 20-inch diameters and held only a few megabytes.
      • Unlike floppy disks, they use hard platters for magnetic storage.
      • Modern hard disks, typically with 40GB capacity, can access data in fractions of a second and are used for frequently accessed data due to their speed.
      • Fixed within computers, hard disks are less portable but some detachable versions exist.
      • They are fragile, susceptible to pressure, jerks, and power fluctuations.
  • Optical Storage Devices: In these storage media, data is stored as light signals, with 0's and 1's converted into light and written by the read-write head. When reading, the bit-pattern of 0's and 1's is generated from the stored light information.
    • CD-ROM:
      • CD-ROM (Compact Disk-Read Only Memory) is an optical disk with spiral pits burned by a laser to store data.
      • It allows random access to information, which can be read from any point on the disk.
      • A standard CD can hold up to 650 MB of data, equivalent to 465 floppy disks.
      • CDs can store text, animation, video, and sound.
      • Write-once CDs store data that cannot be altered, while read/write CD-Rs allow users to write and erase data multiple times with a CD-Writer (CDWR) drive.
    • DVD-ROM:
      • DVDs (Digital Versatile Disks) resemble CD-ROMs but can store between 4.7GB and 17GB of data.
      • They are primarily used for high-quality video and audio storage, with movies commonly stored on them.


Computer Software


Software is a set of instructions enabling hardware to perform tasks. It includes operating systems, applications, and utilities, allowing users to use the computer for specific purposes. Without software, hardware is ineffective, as software enables computers to perform almost any function within their capability.



Types of Software


Computer Software is generally divided into two categories:

  • System Software
  • Application Software


  1. System Software: Includes pre-installed programs and documentation from the computer manufacturer, designed to stay permanently on the machine. Its main goals are to improve hardware efficiency, simplify computer use (primarily through the operating system), interpret and execute application software, and include components like language translators (compilers and interpreters), operating systems, utilities, and specialized programs.

    • Operating Systems: The most important system software that is essential to operate the computer system. It manages:
      • Computer's resources
      • Scheduling multiple jobs for execution
      • Flow of data and instructions between input/output units and the main memory
    • Utility Software: Frequently used in program development. Examples:
      • Sort-merge programs, for organizing records in a specific order
      • Programmers can create their own utility software and store it in secondary memory
    • Programming Software: Used to develop other software, often known as an Integrated Development Environment (IDE). An IDE typically includes a source code editor, a compiler or interpreter, and a debugger.
      • Editor: Used to create and edit application programs. Functions include: entering, searching, deleting, replacing, copying, and inserting text.
      • Compiler: Converts an entire high-level language program (e.g., Pascal, C) into machine language.
      • Interpreter: Translates statements into machine language one line at a time during execution.
      • Assembler: Converts a program in assembly language into machine language.
      • Debugger: Finds and fixes programming errors.
      • Linker: Combines multiple object files generated by the compiler into a single executable program.
      • Loader: Loads and relocates executable programs into main memory, handling storage allocation and ensuring proper execution.
    • Device Driver: Programs that enable higher-level software to interact with hardware. Types include:
      • Logical Device Drivers (LDD): Written by operating system vendors
      • Physical Device Drivers (PDD): Developed by hardware vendors

  2. Application Software: Application software is designed for specific tasks, like accounting or result preparation, and is mostly written in high-level languages.

    • General Purpose Software: Used for tasks like word processing, spreadsheets, and graphic design. Advantages include low cost, easy availability, thorough testing, and ample user support. Examples:
      • Desktop Publishing Software: Used for creating graphics for displays, promotional items, and signs
      • Word-Processing Software: Used for creating, editing, formatting, and printing documents (e.g., letters, reports). Examples:
        • MS Word
        • WordPad
        • LibreOffice Writer
      • Spreadsheet Software: Used for managing and analyzing numerical data. Examples:
        • MS Excel
        • Lotus 1-2-3
        • Apple Numbers
      • Database Software: Organizes and stores data. Examples:
        • Oracle
        • MS Access
        • EasyPhp
      • Graphic Package: Software for creating and editing images or photos. Examples:
        • CorelDraw
        • Paint Shop Pro
      • Presentation Software: Used to create and display slideshows with text, graphics, and other media. Examples:
        • MS PowerPoint
        • Macromedia Director
    • Specific Purpose Software: Highly specialized software designed to handle specific tasks. Examples:
      • Communications Software: Used to send messages and emails
      • Desktop Publishing Programs: Combine and control graphics and text in one document
      • Web Browser: Allows users to access and navigate websites
      • Enterprise Software: Manages organizational processes and data flow
      • Multimedia Software: Allows creation and playing of audio and video files


Programming Languages


Programming languages are classified as lower or higher based on how close they are to the computer's language (machine language, which uses 0s and 1s) or to human languages (higher, which is more English-like).


The main categories of programming languages are:


  • Machine Language: Programs written only in binary digits are called machine language programs. Writing and understanding machine language is challenging.
    • In binary, letters and special characters are represented by sequences of 0s and 1s; for example, the letter A is represented as 1000001.
    • Special characters like commas and semicolons are also represented in binary.
  • Assembly Language: In the 1950s, assembly languages were created to simplify programming by using mnemonic codes instead of binary.
    • These languages are converted to machine code to be executed.
    • Assembly language programs are called source programs, while machine code programs are object programs.
    • Each assembly instruction directly corresponds to a machine code instruction, but macro instructions were introduced to group multiple instructions for efficiency.
  • High-Level Language: High-level languages simplified programming by reducing the need for detailed computer operations, allowing programmers to focus more on the problem-solving logic.


Types of Programming Languages


  • Procedural Programming Language: Focuses on executing a sequence of statements to achieve a result, using variables, loops, and other elements. Functions may control variables and perform tasks like printing.
    • C
    • Pascal
    • Perl
    • PHP
  • Functional Programming Language: Emphasizes functions' return values and discourages side effects and state changes. Avoids loops in favor of recursion.
    • Python
    • Haskell
  • Object-Oriented Programming Language: Treats the world as objects with internal data and methods to access it. Focuses on encapsulation, inheritance, and polymorphism for reusability.
    • Java
    • C++
    • Kotlin
  • Scripting Programming Language: Often procedural and may include object-oriented features, but typically not used for large system development.
    • JavaScript
    • PHP
    • Python
  • Logic Programming Language: Allows declarative statements where the machine deduces consequences based on given rules, instead of specifying exact steps.
    • Mercury
    • Visual Prolog


Their implementation in the different institutions in the community


Computers are deeply integrated into daily life across various sectors:


  • Banks and Financial: Computers manage transactions like ATM withdrawals, digital currency records, and stock trading through advanced algorithms.
  • Business: Computers power registers, employee workstations, and servers to manage transactions, operations, and communications.
  • Communication: Smartphones, e-mail, VoIP, speech assistance, and voice recognition rely on computers for global communication.
  • Defense and Military: Computers are used for encryption, GPS tracking, computer-aided flight, and operating drones.
  • Education: Computers aid in learning through the internet, interactive whiteboards, writing, testing, and tracking student progress.
  • Internet: DNS, web servers, and programs like search engines and FTP services depend on computers for function and accessibility.
  • Medical: Computers store medical records, assist in patient monitoring, research, diagnosis, and even in surgery with robotics.
  • Transportation: Computers control modern cars, traffic lights, GPS navigation, airplanes, public transportation, and self-driving cars.
  • Multimedia: Computers are used in editing, CGI, photo and video manipulation, recording, playback, and creating multimedia content.
  • Robots: Computers control robotics machinery for tasks, learning, and adapting to new conditions.
  • Simulations: Complex problems like weather prediction, product simulations, and big data analysis are solved quickly and accurately using computers.
